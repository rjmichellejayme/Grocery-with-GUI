# 🛒 Grocery Buddy System

A simple Python-based grocery management project that includes a GUI app, a command-line list manager, and a search system. This project is designed to demonstrate basic programming concepts and algorithms.

---

## Features

### Grocery Buddy GUI
- Add items with price and quantity  
- Prevent duplicate entries  
- Remove items from the list  
- Display full grocery list  
- Calculate total cost  
- Built using Tkinter  

---

### Grocery List Manager (CLI)
- Add grocery items  
- Remove items  
- Print full list  
- Menu-driven interface  

---

### Search System
- Linear Search implementation  
- Binary Search implementation  
- Searches numeric values  
- Demonstrates algorithm behavior  

---

## How to Run

### Run GUI App
```bash
python "Grocery Buddy GUI.py"
python grocery_list_manager.py
